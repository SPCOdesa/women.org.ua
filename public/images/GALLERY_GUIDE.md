## Як Додати Нові Зображення

### Крок 1: Підготуйте Зображення

1. Переконайтеся, що розмір відповідає рекомендаціям (350-500px ширина)
2. Оптимізуйте розмір файлу (< 500KB)
3. Збережіть у форматі JPG або PNG

### Крок 2: Завантажте в Папку

Розмістіть файл в `public/images/` з назвою `image-XXX.jpg`

### Крок 3: Оновіть Галерею

Відредагуйте `src/pages/Media.svelte` і додайте новий об'єкт в масив `galleryImages`:

```javascript
const galleryImages = [
  // ... існуючі зображення ...
  {
    src: "/images/image-XXX.jpg",
    alt: "Опис вашого зображення",
  },
];
```

## Поради для Найкращого Результату

### Вибір Фотографій

1. **Різноманітність:** Показуйте різні аспекти роботи організації
2. **Якість:** Використовуйте чіткі, добре освітлені фотографії
3. **Люди:** Фотографії з людьми більш привабливі (з дозволу)
4. **Дія:** Показуйте активність, а не статичні сцени
5. **Емоції:** Вибирайте фотографії, які передають емоції та цінності

### Оптимізація Зображень

## Технічні Деталі

### Компонент Gallery.svelte

**Особливості:**
- Використовує Svelte для реактивності
- Lucide-svelte для іконок
- CSS Grid для адаптивної сітки
- Модальне вікно з backdrop blur
- Клавіатурна навігація

**Розміри сітки:**
- Мобільні (< 640px): 1 колонка
- Планшети (640-1024px): 2 колонки
- Десктоп (> 1024px): 3 колонки

### Стилізація

Галерея використовує CSS змінні для теми:
- `--color-primary` - основний колір
- `--color-secondary` - вторинний колір
- `--color-primaryDark` - темний основний
- `--color-primaryLight` - світлий основний

## Часті Питання

**Q: Як змінити кількість колонок?**
A: Відредагуйте `Gallery.svelte` - змініть `lg:grid-cols-3` на потрібне значення

**Q: Як додати фільтри за категоріями?**
A: Розширте компонент, додавши властивість `category` до кожного зображення

**Q: Як оптимізувати завантаження?**
A: Використовуйте lazy loading або розділіть на сторінки

**Q: Чи можна змінити розмір модального вікна?**
A: Так, відредагуйте `max-w-4xl` в `Gallery.svelte`
