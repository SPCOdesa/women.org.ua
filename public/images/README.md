# Зображення для сайту

Папка для зберігання всіх зображень, які використовуються на сайті.

## Файли, які потрібно додати

### Обов'язкові
- `logo.png` - Логотип організації (200x200px)
- `hero-bg.jpg` - Фонове зображення для hero секції (1920x1080px)
- `board.jpg` - Фото правління (1200x600px)

### Опціональні
- `team-member-1.jpg` - Фото члена команди (300x300px)
- `team-member-2.jpg` - Фото члена команди (300x300px)
- `project-1.jpg` - Зображення проекту (400x300px)
- `project-2.jpg` - Зображення проекту (400x300px)

## Рекомендації

1. **Формат**: JPG для фотографій, PNG для графіки з прозорістю
2. **Розмір файлу**: Не більше 500KB на файл
3. **Оптимізація**: Використовуйте TinyPNG або подібні сервіси
4. **Розширення**: Назви файлів мають бути латиницею, без пробілів

## Як додати зображення

1. Помістіть файл в цю папку
2. Оновіть посилання в компонентах:
   ```svelte
   <img src="/images/logo.png" alt="Логотип" />
   ```
