# ГО Жіночий Центр - Svelte

Веб-сайт громадської організації "Жіночий Центр - Підтримка, Захист та Турбота", переписаний на Svelte.

## Технології

- Svelte 5
- Vite
- Tailwind CSS 4
- Svelte Routing
- Lucide Icons

## Встановлення

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Збірка

```bash
npm run build
```

## Структура проекту

```
src/
```

## Особливості

- Підтримка двох мов (українська/англійська)

- Маршрутизація на стороні клієнта
- Оптимізована збірка

## Система перекладів

Переклади зберігаються в `src/lib/translations.js` та організовані за мовами (uk, en) та сторінками:

### Структура перекладів

```javascript
translations = {
  uk: {
    siteName: 'Назва сайту',
    nav: { /* навігація */ },
    home: { /* домашня сторінка */ },
    about: { /* про нас */ },
    projects: { /* проекти */ },
    media: { /* медіа */ },
    tenders: { /* тендери */ },
    partners: { /* партнери */ },
    contacts: { /* контакти */ },
    footer: { /* підвал */ },
  },
  en: { /* англійські переклади */ }
}
```

### Додавання нових перекладів

1. Додайте ключ в `src/lib/translations.js` для обох мов (uk та en)
2. Використовуйте в компонентах через `$t` store: `{$t.about.hero.title}`
3. Дотримуйтесь структури: `{page}.{section}.{key}`

### Сторінки та їх переклади

- **home** - Домашня сторінка з героєм, принципами та структурою організації
- **about** - Про нас (героїчна секція та опис)
- **projects** - Проекти (героїчна секція)
- **media** - Медіа (героїчна секція)
- **tenders** - Тендери (героїчна секція)
- **partners** - Партнери (героїчна секція)
- **contacts** - Контакти (гаряча лінія та контактна інформація)
